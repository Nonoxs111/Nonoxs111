## Selected Work

### AI Reading Assistant

一款提升技术文献阅读效率的浏览器扩展插件。

* AI Agent 架构设计与实现
* 提示词工程与输出质量优化
* 文章正文智能提取
* 流式响应与实时交互
* 支持追问式深度阅读

React · TypeScript · WXT

<br>

### Mini Markdown Editor

一套可灵活集成的 Markdown 编辑组件库。

* 基于 Monorepo 的多包管理
* 集成 CodeMirror 编辑内核
* 使用 Zustand 进行状态管理
* 支持工具栏按需扩展
* 内置明暗主题切换

React · TypeScript · Vitest

<br>

### Vue3 E-Commerce

一个完整的电商前端项目，涵盖用户认证、购物车、订单管理和支付流程。
